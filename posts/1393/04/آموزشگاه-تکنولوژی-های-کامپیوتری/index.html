آموزشگاه تکنولوژی های کامپیوتری

ارسال شده در Jun 30, 2014

به نام خدا

با سلام و درود بر شما دوست گرامی.

به اطلاع می رسانم که ما به زودی، یک آموزشگاه مجازی راه اندازی خواهیم کرد که در آن دوره های جامع آموزشی در زمینه های زیر تشکیل خواهد شد.

  • تکنولوژی های مایکروسافت
    • برنامه نویسی C#‎
    • برنامه نویسی ویندوز با Windows Form applications
    • برنامه نویسی ویندوز با WPF و یا Windows Presentaion Foundation
    • Microsoft Linq
    • برنامه نویسی وب با Mcrosoft ASP.NET Web Forms
    • برنامه نویسی وب با Microsoft ASP.NET MVC
    • آموزش Microsoft Entity Framework
    • دیتابیس Microsoft SQL Server 2012
    • و ...
  • تکنولوژی های عمومی وب
    • HTML
    • CSS
    • Javascript
    • jQuery
  • تکنولوژی های متن باز
    • Linux
    • Web Server
    • Linux Security
    • MySQL
    • Hadoop
  • آموزش های تخصصی دروپال و ملحقات آن
    • مقدمات دروپال
    • دروپال پیشرفته
    • ماژول نویسی دروپال
    • طراحی پوسته برای دروپال
    • توسعه ی دروپال
    • تسلط به ماژول های دروپال
  • زبان های برنامه نویسی
    • PHP
    • C++‎
    • Erlang
    • و ...

این دوره های آموزشی توسط مهندسین خبره در هر حوزه تهیه می گردد. و به صورت فیلم های آموزشی با کیفیت بالا و همچنین آموزش حضوری (خصوصی یا عمومی) برگزار خواهد گردید.

مژده این که اکثر آموزش های مرتبط با محصولات مایکروسافت هم اکنون آماده می باشند. و به زودی در اختیار شما قرار خواهد گرفت

مژده ی دیگر این که بسیاری از آموزش ها رایگان خواهد بود. و باقی نیز با هزینه ی اندک و بسیار خوبی ارائه خواهد شد.

به عنوان حسن ختام، آخرین مژده این که امکان خرید بسته های آموزشی در قالب CD یا DVD همراه با تحویل پستی، نیز فراهم خواهد شد.

+ پ.ن. : سایت گروه آموزشی طلوع راه اندازی شد.
