پیدا کردن فایل های بزرگ تر از 100 کیلو بایت

ارسال شده در Sep 17, 2011

برنامه ی زیر یک مثال از کار با فایل ها در VBScript است. به کنک این برنامه می توانید فایل هایی که از جد مشخصی حجمشان بالا تر است را پیدا کنید.

 1Const MAX_SIZE = 1024
 2Dim Fso, File, Folder, Path, Message
 3Message = ""
 4Path = "C:\\Windows"
 5Set Fso = CreateObject("Scripting.FileSystemObject")
 6Set Folder = Fso.GetFolder(Path)
 7For Each File In Folder.Files
 8  If File.Size > MAX_SIZE Then
 9    Message = Message & File.Size & " Bytes =====> " & File.Path & vbCrLf
10  End If
11Next
12MsgBox Message
13Set Folder = Nothing
14Set File = Nothing
15Set Fso = Nothing

برنامه ی فوق را می توانید از این جا دانلود کنید:  [FinLargeFiles.zip]

فقط در موقع اجرای برنامه دقت کنید که ممکن است یک لیست طولانی از نام فایل ها را برگرداند.
